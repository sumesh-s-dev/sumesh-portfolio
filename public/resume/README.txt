This directory contains the following resume files:

1. 'sumesh-resume-ats.html' - An ATS-friendly HTML version of the resume that is properly formatted for ATS systems.
2. 'sumesh-resume.html' - A deprecated HTML version of the resume (kept for reference).

To create a proper PDF version of the resume:
1. Open the 'sumesh-resume-ats.html' file in a web browser
2. Use the browser's print functionality (Ctrl+P or Cmd+P)
3. Select "Save as PDF" as the destination
4. Save the file as 'sumesh-resume.pdf' in this directory

The download button on the resume page is configured to download the ATS-friendly HTML file as a PDF.
